# Hope

Some small patches for [Hope](https://en.wikipedia.org/wiki/Hope_(programming_language)) to make it work on my machine(s) (with the main **HOPE** of teaching @vmxdev some functional programming).

You can build it with `make` and run REPL with `make run`.

The original sources can be found [here](http://web.archive.org/web/20070930155415/http://www.soi.city.ac.uk/~ross/Hope/)
